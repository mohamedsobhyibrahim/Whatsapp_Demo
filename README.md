# Whatsapp Demo

Whatsapp demo is a simple chat application like Whatsapp, you can start conversation with any person by sending a chat request

# Tools

- RelativeLayout
- LinearLayout
- drawable Xml Shapes
- CircleImageView
- Firebase Auth
- Firebase Database
- Firebase Storgae
- Firebase Cloud Messaging
